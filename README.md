# Pear Admin Boot   :tada: 

#### 项目介绍  :jack_o_lantern: 

Spring Boot Security MyBatis Thymeleaf 后端权限流程脚手框架

#### 模块划分

|  系统权限   | Security    |  流程控制    |  Activiti    |
| --- | --- | --- | --- |
|  定时任务   | quartz    |  接口文档   |  Swagger   |
|  数据操作   | My Batis   |  核心框架   | Spring Boot     |
